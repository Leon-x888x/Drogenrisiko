# Drogenrisiko
Drogenrisiko ist ein Survival-Brettspiel für 2 bis 6 Spieler, das Verkehrssicherheit und die Risiken
von Suchtmitteln im Straßenverkehr spielerisch vermittelt. Es eignet sich besonders für den
Einsatz in Workshops und schulischen Veranstaltungen.
Das Spiel kombiniert ein klassisches rundenbasiertes Brettspiel mit interaktiven Minispielen, die
über eine Android-App gesteuert werden. Optional kann ein physisches LED-Spielbrett per
Bluetooth verbunden werden, das das Spielerlebnis durch visuelle Effekte deutlich bereichert.
